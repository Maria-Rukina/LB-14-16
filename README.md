# Реализованные функции

1. double* full_elements(double* ptr_array, int n)

* Назначение: заполнение массива значениями
* Элементы: ptr_array - указатель на массив, n - размер массива
* Возвращает: указатель на заполненый массив

2. double func(double x)

* Назначение: функция для заполнения значениями массив
* Элементы: x - вводимый x
* Возвращает: полученный x

3. int put_elements(double* ptr_array, int n)

* Назначение: вывод массива
* Элементы: ptr_array - указатель на массив, n - размер массива
* Возвращает: напечатанный массив и его размер

4. double* calc_elements(double* ptr_array, int n)

* Назначение: умножение каждого элемента массива на -1
* Элементы: ptr_array - указатель на массив, n - размер массива
* Возвращает: указатель на изменёный массив

5. double sum_elements(double* ptr_array, int begin, int end)

* Назначение: сумма элементов массива в диапазоне
* Элементы: ptr_array - указатель на массив, begin - начало диапазона, end - конец диапазона
* Возвращает: результат суммы

6. int find_element(double* ptr_array, int n, double element)

* Назначение: поис элемента
* Элементы: ptr_array - указатель на массив, n - размер массива, element - искомый элемент
* Возвращает: сам элемент, если он найден, или -1 если такого элемента нет

7. int delete_k(double* ptr_array, int n, int k)

* Назначение: удаление элемента по введёному номеру
* Элементы: ptr_array - указатель на массив, n - размер массива, k - номер удаляемого элемента
* Возвращает: размер массива
